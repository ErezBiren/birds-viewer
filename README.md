# birds-viewer
### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/ErezBiren/birds-viewer.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or

   ```sh
   yarn
   ```



### Usage
   ```sh
   npm run dev
   ```
   or

   ```sh
   yarn dev
   ```

    navigate with the browser to http://localhost:5173/
