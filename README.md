![image](https://github.com/ErezBiren/birds-viewer/assets/7828909/c5a801e1-cb35-4c40-9dcf-317bec75bada)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ErezBiren/birds-viewer.git
   ```
2. Install NPM packages

   ```sh
   npm install
   ```

   or

   ```sh
   yarn
   ```

### Usage

```sh
npm run dev
```

or

```sh
yarn dev
```
